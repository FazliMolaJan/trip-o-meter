Trip 'O Meter

An Open Source Android App

Built with Android Studio

"Trip O Meter" is a modern Android app that helps you with fuel prices when you plan for a trip. The app is Ad free and the source code is open source.


Features

The Trip 'O Meter has the below features:


  • Calculate fuel quantity and amount required for a trip
  • Get fuel prices - Coming Soon (In India only)

Fuel Prices API is being developed by RaghuChandrasekaran.
